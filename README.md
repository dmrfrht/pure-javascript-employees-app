# Pure JavaScript Employees App

* Installation
1. You must install json-server on your computer globally. 
1. We run the command `sudo npm instal -g json-server` from the terminal.
1. Then, by going into the project from a terminal and running the `npm install` command, install the project dependencies.
1. Run the command `npm run start` using the same terminal and launch the project.
1. To run json-server, open a different terminal and run the command `json-server --watch fake-api / db.json` there.

* Project URL: `http://localhost:3001`
* json-server URL: `http://localhost:3000/employees`
